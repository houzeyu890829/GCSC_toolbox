# GCSC_toolbox
Matlab code for efficient source identification and parameter estimation of DNAPL-contaminated aquifer using Sensitivity-based Dynamic PSO with Metropolis Criterion and Genetic evolution-kernel extreme learning machine In the present tool box, A parallel global searching strategy in Bayesian inversion framework was designed for simultaneous identification of dense non-aqueous phase liquids (DNAPLs) source characteristics and contaminant transport parameters in groundwater. Genetic evolution-wavelet kernel extreme learning machine (GE-WKELM) was applied to approximate the forward and inverse input-output relationship of the DNAPLs transport numerical simulation model. The adaptive inverse-KELM was established for estimating the source characteristics and contaminant transport parameters preliminarily, thereby correcting the prior information and generating the initial starting points of parallel searching, while the local accurate forward-KELM surrogate model of numerical model was embedded in the searching system for avoiding repetitive CPU-demanding likelihood evaluations. The sensitivity-based dynamic particle swarm optimization (SD-PSO) algorithm with Metropolis criterion was developed for improving the search ergodicity and realizing the precise inversion of all the unknown variables with dramatically varied contribution to the likelihood function.   The main program was named 'P_S_PSO.m' in the folder of 'SD-PSO with Metropolis Criterion'. The established forward GE-WKELM surrogate model had 12 input variables and 8 output variables, and 200 training samples and 20 testing samples were listed in 'Syangben.txt'. The inverse GE-WKELM surrogate model had 8 input variables and 12 output variables.
